1. Copy everything below into Microsoft Word
2. Keep all {variable} placeholders and conditional tags exactly as written
3. Save as template.docx in document-service folder

- {#is_lease}...{/is_lease} - Shows only for lease transactions
- {#is_purchase}...{/is_purchase} - Shows only for purchase transactions

NOTE ON CONDITIONAL TAGS:
- {#is_lease}...{/is_lease} - Content only shows when transaction_type contains "lease"
- {#is_purchase}...{/is_purchase} - Content only shows when transaction_type contains "purchase"

- docxtemplater will process them automatically during generation

FORMATTING GUIDE:
1. Title: Center, Bold, 16pt
2. Section headers (1, 2, 3...): Bold, Underline, 12pt
3. Body text: 11pt, Justified
4. Date: Right-aligned at top
5. Signature lines: Leave 3-4 blank lines above
6. Footer: Small, Italic, Gray
